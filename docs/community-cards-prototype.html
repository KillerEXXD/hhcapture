Card Size & Color Guidelines

Card Dimensions:

  • • Active card: 80px × 112px (w-20 h-28)
  • • Previous cards: 64px × 96px (w-16 h-24)
  • • Rank size: text-4xl (2.25rem)
  • • Suit size: text-4xl (2.25rem)

Suit Colors (Consistent):

  • ♠ Spades: text-gray-900
  • ♥ Hearts: text-red-600
  • ♦ Diamonds: text-blue-600
  • ♣ Clubs: text-green-700

Implementation Notes:

  • ✓ Cards are clickable for manual selection (if auto-select is disabled)
  • ✓ Hover effect shows card label (Flop 1, Turn, River, etc.)
  • ✓ Active card has highlighted border matching street color (Green=Flop, Orange=Turn, Purple=River)
  • ✓ Previous street cards shown at 80% size with reduced opacity for context
  • ✓ Auto-select: When "Auto-select cards" is enabled in setup, community cards are randomly generated
  • ✓ Consistent suit colors across all streets and player hole cards
